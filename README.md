# Bellabeat-Consumer-Growth-Usage-Analysis
An end-to-end data analysis project focused on Bellabeat’s smart device ecosystem (Leaf, Time, Spring). This case study analyzes user trends in activity, sleep, and hydration to identify high-growth marketing opportunities and improve global consumer engagement.
## Goal
The primary objective of this project is to analyze smart device fitness data from Bellabeat users to identify patterns in activity, sleep, and hydration. By uncovering these behavioral trends, the project aims to provide actionable, data-driven marketing recommendations to executive leadership to drive global customer acquisition and improve user retention across the Bellabeat ecosystem (Leaf, Time, and Spring)
## Data Source
FitBit Fitness Tracker Data (CCO:Public Domain): This Kaggle data set contains personal fitness tracker from thirty fitbit users. Thirty eligible Fitbit users consented to the submission of personal tracker data, including minute-level output for physical activity, heart rate, and sleep monitoring. It includes information about daily activity, steps, and heart rate that can be used to explore users’ habits.
https://www.kaggle.com/datasets/arashnic/fitbit
